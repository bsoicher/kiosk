

## Requirements
- [git](https://git-scm.com/)
- [nodejs](https://nodejs.org/)

## Installation

    cd Desktop
    git clone https://github.com/bsoicher/kiosk.git kiosk
    cd kiosk

    npm install
    npm run build
    npm run start

## Updating

    git pull
    npm update
    npm run build
    npm run start
